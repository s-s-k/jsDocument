# ECMAScript 函数概述
### 什么是函数
函数是一组可以随时运行的语句。  
函数通过这样方式声明：关键字function、函数名、参数、以及待执行代码。  
基本语法：  
<pre><code>
function functionName(arg0, arg1,... argN){  
  statuments  
}
</code></pre>
函数可以通过函数名加上括号中的参数进行调用，例如：
<pre>
<code>
say("Ravid", "hello")
</code>
</pre>
函数返回值： 函数通过return语句来返回值。当没有返回值时，默认返回undefined。
